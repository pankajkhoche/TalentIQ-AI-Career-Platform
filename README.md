# TalentIQ - AI Career Intelligence Platform

TalentIQ is a full-stack AI-powered career readiness platform built using FastAPI, PostgreSQL, React, JWT Authentication, and OpenAI integration.

## Features

- User Register/Login
- JWT Authentication
- Resume Upload
- PDF Resume Parsing
- Skill Extraction
- ATS Score Calculation
- AI Resume Review
- Technical Question Generation
- Interview Practice
- Answer Evaluation
- Skill Gap Analysis
- Roadmap Generator
- Job Readiness Score
- Resume History
- Interview History
- Resume Compare
- Analytics Dashboard
- Career Advisor
- Job Matcher

## Tech Stack

### Backend
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT
- OpenAI API

### Frontend
- React
- Vite
- Axios
- React Router
- Recharts
- React Icons
