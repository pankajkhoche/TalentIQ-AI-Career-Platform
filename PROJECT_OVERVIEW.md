# TalentIQ

TalentIQ is an AI-powered Career Intelligence Platform.

Features:

- Resume Analysis
- ATS Scoring
- AI Resume Review
- Interview Practice
- Skill Gap Analysis
- Learning Roadmap
- Readiness Scoring
- Career Advisor
- Job Matcher
- Analytics Dashboard
- Resume History
- Interview History
- Resume Comparison

Tech Stack:

Backend:
- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT

Frontend:
- React
- Vite
- Axios
- Recharts

Deployment:
- Docker
